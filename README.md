# X Login UI Clone

Reproduction de l'écran de connexion de X (anciennement Twitter) réalisée avec Flutter dans le cadre d'un projet d'apprentissage et de pratique du développement d'interfaces utilisateur.

## Aperçu

Ce projet vise à recréer l'interface de connexion de X en utilisant Flutter tout en appliquant les bonnes pratiques de développement et de conception d'interfaces.

## Fonctionnalités

* Interface inspirée de X
* Design moderne et épuré
* Mise en page responsive
* Structure de code claire et maintenable
* Compatible Android, iOS et Web (selon la configuration Flutter)

## Technologies utilisées

* Flutter
* Dart

## Installation

### Cloner le dépôt

```bash
git clone https://github.com/Mourtada-002/X-LoginUI.git
```

### Accéder au dossier du projet

```bash
cd X-LoginUI
```

### Installer les dépendances

```bash
flutter pub get
```

### Lancer l'application

```bash
flutter run
```

## Objectifs du projet

Ce projet m'a permis de travailler sur :

* La création d'interfaces avec Flutter
* La gestion des layouts et de l'affichage responsive
* L'organisation du code en composants réutilisables
* La reproduction d'une interface populaire à des fins éducatives
